Concepts

Binding

When Rava is instantiated it creates an empty object to maintain the mappings between the css selector and the associated configuration object.

There is only one configuration tied to a sepecific selector. If you use the same selector with a different configuration, the new one will overwrite the old

The method to register a css string and it's configuration object is the bind method.

Event Handling

Event registering is done using the events subconfiguration in the configuration object. There are two types of configuration done.
