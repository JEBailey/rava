Method Declaration

Unlike other options, the method configuration doesn't support a scoped method of declaration. All methods declared are mapped directly onto the primary binding. The traditional way of declaring functions is to use the methods Object 
            {
                methods : {
                    basicFunctionDecl : function() {
                        //do stuff
                    }
                }
            }
To reduce depth the configuration object now supports direct mapping of any functions that are not keyed to a specific feature 
            {
                basicFunctionDecl : function(){
                    //do stuff
                }
            }
