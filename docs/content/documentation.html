bind

configuration assignment

Rava uses the bind method to associate a configuration object with all elements on a page that matches the CSS selector string

In addition, the configuration is registered so that any elements that are later added to the page will be processed with the same configuration

find

scoped search for first match by CSS selector

find traverses the children of the given element and finds the first matching element

findAll

scoped search for all matches by CSS selector

findAll traverses the children of the given element and finds all of the matching elements
