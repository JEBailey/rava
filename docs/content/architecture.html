Architecture

decorate

signature

Rava.decorate(selector,ravaConfigObject)

parameters

selector

The first parameter for the decorate method is a css selector string. This is used to identify whether any element, either current or future, are elements that we should decorate.

Examples of valid selectors are;

  • "#nav li"
  • ".burger"
  • "rava-time"

ravaConfigObject

The default config Object is

    {
        callbacks: {
            created:
            added:
        },
        methods: {
            methodName: functionDefinition(){}
        },
        events:{
            eventName: functionDefinition(){},
            selector: {
                eventName : functionDefinition(){}
            }
        },
        data: {}
    }
