References

A reference is where we store a reference to one or more elements to the data object for a specific rava binding. A reference is specified using the refs configuration 
            {
                // stores a reference to the current element as data with the key 'foo'
                // i.e. data.foo
                refs : "foo"
            }
By using an object in the configuration you can store multiple references 
            {
                refs : {
                    ".foo" : "foo",
                    ".bar" : "bar",
                }
            }
